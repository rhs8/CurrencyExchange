CurrencyExchange

Iran FX Watch — tracks the Iranian rial (IRR) against major currencies, compares official (CBI) vs market (Bonbast), and surfaces alerts.

This GitHub Pages site is only a landing page. GitHub cannot run your database, ingest worker, or FastAPI server. That is expected.

The real dashboard is the api service at / after you run Docker locally or deploy the stack (Railway / Render / Fly.io).

Jump to your deployed API (optional)

Paste the public base URL of your deployed API (must start with http:// or https://), then press Enter.

Example bookmark: ?api=https://YOUR-RAILWAY-OR-RENDER-URL
