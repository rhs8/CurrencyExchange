CurrencyExchange

Iran FX Watch — tracks the Iranian rial (IRR) against major currencies, compares official (CBI) vs market (Bonbast), and surfaces alerts.

You are on the static project page (GitHub Pages / hosted HTML). The live charts and API run as a Docker stack, not inside this file.

  • Source code is on GitHub in this repository.
  • Dashboard is the FastAPI app at / when you run docker compose up or deploy the compose stack to Railway / Render / Fly.io.
View repository Quickstart (Docker)

Run locally

git clone https://github.com/rhs8/CurrencyExchange.git
cd CurrencyExchange
docker compose up -d --build
# then open http://localhost:8000
